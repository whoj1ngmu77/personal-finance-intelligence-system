# 💰 Personal Finance Intelligence System using Predictive Analytics

An AI-powered financial analytics platform that transforms raw transaction data into actionable financial intelligence through analytics, behavioral insights, risk detection, and predictive modeling.

## 🚀 Overview

Managing personal finances is often difficult due to fragmented spending records, impulsive expenses, and lack of visibility into financial behavior.

This project solves that problem by allowing users to upload raw financial transaction data (bank statements, UPI history, GPay, PhonePe, Paytm exports, CSV/XLSX files) and automatically generate:

* Spending analytics
* Behavioral insights
* Overspending detection
* Financial health scoring
* Smart recommendations
* Predictive spending analysis

The goal is to simulate a real-world fintech intelligence system rather than a basic expense tracker.

## ✨ Features

### 📂 Upload-Based Analytics

* Upload CSV or Excel transaction files
* Supports raw real-world financial datasets
* Handles varying dataset sizes

### 🧹 Automatic Data Cleaning

* Missing value handling
* Duplicate removal
* Date standardization
* Merchant normalization
* Data preprocessing

### 📊 Interactive Analytics Dashboard

* Spending by category
* Merchant-wise spending
* Payment method distribution
* Daily/monthly spending trends
* Time-based analytics

### 🧠 Behavioral Intelligence

* Late-night spending detection
* Impulsive purchase analysis
* Weekend overspending insights
* High-value transaction tracking

### 🤖 Predictive Analytics

* Spending prediction model
* Overspending risk prediction
* Financial behavior analysis

### 💡 Smart Recommendations

Personalized recommendations based on spending behavior.

Examples:

* Reduce late-night spending
* Set shopping budget limits
* Monitor overspending patterns



